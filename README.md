# 🎓 College Discovery Platform

An AI-powered modern college discovery and prediction platform built using Next.js, TypeScript, Tailwind CSS, Node.js, Express, and Prisma.

The platform helps students:

- Discover top colleges
- Predict colleges based on rank
- Explore placement statistics
- View detailed college profiles
- Register/Login securely
- Experience a modern responsive UI

---

# 🚀 Features

## 🔐 Authentication System

- User Registration
- User Login
- JWT Token Authentication
- Secure API Handling

## 🤖 AI Rank Predictor

- Predict colleges using exam rank
- Supports JEE and NEET
- Dynamic prediction cards
- Responsive modern UI

## 🏫 College Discovery

- Browse colleges
- Search and filter colleges
- Dedicated college detail pages
- Placement information
- Courses offered
- Ratings and fees

## 🎨 Modern UI/UX

- Dark premium design
- Glassmorphism cards
- Smooth hover animations
- Fully responsive layout
- Gradient backgrounds
- Mobile-friendly interface

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/techkinha553-source/College-Platform
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

# ▶️ Run Project

## Start Backend

```bash
cd backend
npm run dev
```

Backend runs on:

```bash
http://localhost:5050
```

---

## Start Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on:

```bash
http://localhost:3000
```
